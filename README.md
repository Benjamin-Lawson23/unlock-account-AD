<h1>Unlocking an Account and Resetting a Password in Active Directory</h1>

This is brief video demonstration on how to unlock a locked account and reset its password in Active Directory. Click the link below to watch.<br />

<h2>Environments and Technologies Used</h2>

- Microsoft Azure (Virtual Machines)
- Remote Desktop
- Active Directory Domain Services

  <h2>Video Demonstration</h2>

- ### [YouTube: Basic Subnetting Demo](https://youtu.be/iC6TNmgU1Us)

<h2>Summary of actions</h2>

<p>
  
  1. Navigate to 'Active Directory Users and Computers'
  2. Find the correct Organizational Unit
 
  <p>
<img src="https://imgur.com/V943Tij.png" width="40%" alt="Disk Sanitization Steps"/>
</p>
<p>  
  
  3. Find the account that needs to be unlock and double click on it
  4. Go to the Account tab
  5. Check the 'Unlock account' box

  
  <p>
<img src="https://imgur.com/V943Tij.png" width="40%" alt="Disk Sanitization Steps"/>
</p>
<p>  
  
  6. Right-click on the account name
  7. Click on Reset Password
 
  <p>
<img src="https://imgur.com/V943Tij.png" width="40%" alt="Disk Sanitization Steps"/>
</p>
<p>  
  
  8. Enter a new password
 
  
FINISHED
   </p>
